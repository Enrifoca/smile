# Security Policy

## Supported Versions

The following versions of smile:D are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in smile:D, please report it privately by emailing **enrifoca@gmail.com**.

Please include:
- A description of the vulnerability
- Steps to reproduce it
- Affected versions
- Any possible mitigations

We will respond as soon as possible and coordinate a fix and disclosure.

Please do **not** open public issues for security vulnerabilities.
