## Brave Search connector

- Use `brave_web_search` for current events, documentation, or facts not available in the workspace.
- Always cite source URLs when summarizing results.
- Prefer concise answers in chat. If the search returns many results or the answer needs structured formatting, use `report_write` instead.
- If the API returns a 401 error, tell the user to check their Brave Search API key in Connectors settings.
